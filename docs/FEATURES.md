# Features

## MVP

### Prompt Editor

- Monaco editor
- Line numbers
- Search
- Dark theme
- Token/character estimate
- Risk phrase highlights

### Model Selection

- Static model catalog
- Multi-select checkboxes
- Select all and clear actions
- Latest status per model

### Run Settings

- User message
- Max tokens
- Temperature
- Streaming toggle
- Include tools toggle

### Test Runs

- Run selected models
- Show loading state per model
- Capture status, finish reason, token usage, latency, and response ID
- Persist run and results

### Results Matrix

- Sortable table
- Expandable raw details
- Copy response
- Clear visual state for content filter

### Version History

- Auto-save prompt snapshot per run
- Manual Save Version
- Restore version
- Version timeline

### Diff Viewer

- Compare current prompt to selected version
- Highlight added and removed lines
- Highlight risky changed phrases

### Risk Inspector

- Deterministic scan
- Line-linked findings
- Confidence and category

## Post-MVP

- Live streaming updates
- Run cancellation
- Model presets
- Prompt projects
- Raw request capture import
- Tool schema editor
- Provider credential settings UI
- Export run report as markdown or JSON
- Shareable local report bundle
- A/B prompt comparison
- Automated binary search over prompt sections
- Suggested safe rewrites

## Stretch Features

- Multi-provider benchmark runs
- Organization/team collaboration
- Cloud sync
- Scheduled regression tests
- CI integration
- Browser extension for importing captured requests
